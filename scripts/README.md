# Scripts

Scripts de automatización para el ciclo de vida del curso.

## Disponibles

- [`bootstrap-repo.sh`](./bootstrap-repo.sh) — Inicializa el repo en GitHub la primera vez

## Pendientes (futuras fases)

- `build-pdfs.py` — Genera un PDF por módulo a partir de los markdown fuente
- `build-web.py` — Genera el e-learning HTML interactivo a partir de los markdown fuente
- `validate.py` — Valida consistencia, enlaces rotos, formato de los markdown del curso
- `deploy-pages.sh` — Despliega el e-learning a GitHub Pages
