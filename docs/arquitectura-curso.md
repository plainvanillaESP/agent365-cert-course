# Arquitectura del curso

> Diseño maestro detallado de los 17 módulos. Este documento se completa durante la **Fase 1** del plan y se valida antes de empezar la producción de contenido.

**Estado:** 🟡 En curso (Fase 1)

---

## Visión

Al finalizar el curso, el alumno demuestra competencia en:

1. Comprender el rol de Microsoft Agent 365 dentro del ecosistema de gobernanza de IA de Microsoft
2. Configurar el tenant para gestionar agentes de manera segura
3. Implementar identidades, accesos y políticas de Conditional Access para agentes
4. Aplicar protección de datos con Microsoft Purview a flujos de agentes
5. Monitorizar, auditar y responder a amenazas con Microsoft Defender
6. Aplicar políticas de gobernanza coherentes entre Microsoft Agent 365, Copilot Control System y plataformas creadoras (Copilot Studio, Foundry, SharePoint)
7. Gestionar costes y optimizar la inversión en agentes
8. Diagnosticar problemas comunes en la operación de agentes
