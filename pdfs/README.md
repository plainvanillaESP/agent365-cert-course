# PDFs

Esta carpeta contendrá los PDFs descargables generados a partir del contenido fuente en markdown.

**Estado:** Pendiente — se construye en la Fase 8 del plan.

## Outputs previstos

- `modulo-01.pdf`, `modulo-02.pdf`, ..., `modulo-17.pdf` — un PDF por módulo
- `manual-completo.pdf` — manual del curso completo (todos los módulos en un solo PDF)
- `examen-certificacion.pdf` — examen imprimible para sesiones presenciales

## Generación

```bash
python scripts/build-pdfs.py
```

(El script se desarrollará en la Fase 8.)
