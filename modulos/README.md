## Estructura estándar de un módulo

Cada carpeta de módulo contiene:

- `README.md` — Índice del módulo y duración
- `teoria.md` — Contenido teórico
- `laboratorios.md` — Laboratorios prácticos paso a paso
- `evaluacion.md` — Preguntas de comprobación + caso de estudio
- `recursos.md` — Referencias a documentación oficial
- `assets/` — Capturas, diagramas y otros recursos visuales

El módulo 17 tiene una estructura ligeramente distinta porque es el examen final.
