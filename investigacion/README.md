## Política de actualización

El producto Microsoft Agent 365 evoluciona muy rápido. La investigación base debe revisarse cada 3 meses, especialmente:

- Nuevas capacidades GA que estaban en preview
- Nuevos roles administrativos
- Cambios en pricing
- Nuevas integraciones (registry sync con más plataformas, etc.)
- Cambios en APIs de Microsoft Graph
- Nuevos comandos del Agent 365 CLI

Cuando se realice una nueva investigación o actualización, crear un nuevo archivo `deep-research-YYYY-MM.md` y actualizar el changelog del repositorio.
