# Módulo 14 — Gobernanza avanzada y multi-tenant

**Duración estimada:** 60 min lectura + 12 min quiz + 60 min labs = 132 min total.
**Área del examen:** 5 (Monitor, troubleshoot and improve).
**Estado:** ✅ Producido (Fase E.8).

Aporta **2 preguntas al banco oficial** del examen.

## Archivos del módulo

- [`teoria.md`](./teoria.md) — Contenido teórico (5 secciones, 60 min lectura).
- [`quiz-practica.md`](./quiz-practica.md) — 5 preguntas Q-14-1..Q-14-5.
- [`laboratorios.md`](./laboratorios.md) — 2 labs prácticos (60 min total).

## Objetivos de aprendizaje

Al finalizar el módulo el alumno será capaz de:

1. Diferenciar las 4 topologías multi-tenant (parent/subsidiary, M&A, MSP/MSSP, joint venture).
2. Configurar delegated administration cross-tenant con GDAP + Customer Lockbox.
3. Diseñar políticas cross-tenant que respeten regulación local.
4. Implementar los 3 federation models (centralizado, federado, hub-and-spoke).
5. Operacionalizar el ciclo de gobernanza distribuida.

## Prerrequisitos

- M01 (fundamentos), M06 (Entra Agent ID), M09 (Conditional Access), M12 (Defender XDR), M13 (CCS).
- Familiaridad con regulaciones aplicables (GDPR, CCPA, LGPD, etc.).
