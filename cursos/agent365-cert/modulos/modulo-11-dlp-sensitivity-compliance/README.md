# Módulo 11 — DLP, sensitivity labels avanzadas y Communication Compliance

**Duración estimada:** 90 min lectura + ~30 min quiz y labs (cuando se produzcan).
**Área del examen:** 4 (Implement data protection with Microsoft Purview, peso 20 %).
**Estado:** En producción. Teoría completada en Fase E.3; quiz de práctica y laboratorios pendientes en Fase E.4.

## Archivos del módulo

- [`teoria.md`](./teoria.md) — Contenido teórico completo (7 secciones, 90 min lectura).
- [`recursos.md`](./recursos.md) — URLs de Microsoft Learn, blogs oficiales y lecturas adicionales.
- `quiz-practica.md` — *(pendiente Fase E.4)* — preguntas Q-11-N de práctica.
- `laboratorios.md` — *(pendiente Fase E.4)* — 4 labs prácticos.
- `module.yaml` — manifiesto del módulo con OAs, glosario y secciones.

## Objetivos de aprendizaje

Al finalizar el módulo el alumno será capaz de:

1. Diferenciar DLP de Information Protection (M10).
2. Diseñar la anatomía completa de una DLP policy para outputs de agentes.
3. Entrenar y calibrar trainable classifiers custom.
4. Configurar DLP runtime para outputs de agentes con flujo de justificación.
5. Integrar DLP con Microsoft Defender for Cloud Apps para SaaS de terceros.
6. Aplicar Communication Compliance a conversaciones con participación de agentes.
7. Operacionalizar el conjunto DLP + CC + Insider Risk.

## Prerrequisitos

- Módulo 01 (fundamentos de Agent 365).
- Módulo 09 (Permisos y Conditional Access).
- Módulo 10 (Purview e Information Protection).
