# Módulo 13 — Copilot Control System integrado con Agent 365

**Duración estimada:** 45 min lectura + 12 min quiz + 60 min labs = 117 min total.
**Área del examen:** 5 (Monitor, troubleshoot and improve).
**Estado:** ✅ Producido (Fase E.7).

Módulo compacto que cierra la gobernanza ejecutiva del programa Agent 365. Aporta **1 pregunta al banco oficial** (la más baja del Área 5, reflejo del peso temático).

## Archivos del módulo

- [`teoria.md`](./teoria.md) — Contenido teórico (5 secciones, 45 min lectura).
- [`quiz-practica.md`](./quiz-practica.md) — 5 preguntas Q-13-1..Q-13-5.
- [`laboratorios.md`](./laboratorios.md) — 2 labs prácticos (60 min total).
- [`recursos.md`](./recursos.md) — URLs de Microsoft Learn y referencias.
- `module.yaml` — manifiesto del módulo con 5 OAs, glosario y secciones.

## Objetivos de aprendizaje

Al finalizar el módulo el alumno será capaz de:

1. Posicionar CCS como panel unificado de gobernanza para Copilot y agentes.
2. Identificar las 4 superficies operativas del portal.
3. Configurar políticas centralizadas (allowlist/blocklist, approval flows).
4. Diferenciar CCS de Defender XDR y Purview.
5. Operacionalizar CCS en el ritual semanal + reporte mensual + revisión trimestral.

## Prerrequisitos

- M01 (fundamentos de Agent 365).
- M06 (Microsoft Entra Agent ID).
- M09 (Permisos y Conditional Access).
- M12 (Microsoft Defender XDR).

Conocer el rol de Defender XDR y Purview es prerrequisito para entender la diferenciación operativa con CCS.
