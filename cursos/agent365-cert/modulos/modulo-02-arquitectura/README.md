**Duración:** 75 minutos · **Fase:** F3 · **Dependencias:** M01 · **Estado:** producido

## Resumen

Diagrama de bloques completo de Microsoft Agent 365 y sus integraciones: Microsoft 365 admin center (Agent workload, Registry, Map, Overview), Microsoft Entra Agent ID, Microsoft Purview, Microsoft Defender XDR, Work IQ MCP servers y los dos SDKs.
