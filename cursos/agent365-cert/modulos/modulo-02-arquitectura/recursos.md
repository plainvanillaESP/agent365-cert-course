## Documentación oficial de Microsoft

### Agent workload en Microsoft 365 admin center

- [Microsoft Agent 365 overview](https://learn.microsoft.com/microsoft-agent-365/) — punto de entrada general del producto.
- [Agent 365 in M365 admin center — Overview](https://learn.microsoft.com/microsoft-365/admin/manage/agent-365-overview) — explicación de la página Overview y de las cuatro hero metrics.
- [Manage the agent registry](https://learn.microsoft.com/microsoft-365/admin/manage/manage-agent-registry) — Registry: filtros, columnas, página de detalle.
- [Manage the agent map](https://learn.microsoft.com/microsoft-365/admin/manage/manage-agent-map) — Map: visualización de grafo, agrupación por plataforma.
- [Agent settings in the admin center](https://learn.microsoft.com/microsoft-365/admin/manage/agent-settings) — Settings: integraciones con Defender, Purview, defaults de publishing.

### Agent identities (Entra)

- [What is Microsoft Entra Agent ID](https://learn.microsoft.com/entra/agent-id/identity-platform/what-is-agent-id) — los 4 tipos de objetos del directorio.
- [What are agent identities](https://learn.microsoft.com/entra/agent-id/what-are-agent-identities) — visión técnica del modelo de identidad.

### Tipos de agentes gestionables

- [Connect existing agents to Agent 365](https://learn.microsoft.com/microsoft-agent-365/connect-existing-agents) — qué tipos hay y cómo se registran.
- [Microsoft Copilot Studio overview](https://learn.microsoft.com/microsoft-copilot-studio/) — DA, CEA y Backend Plugin.
- [Microsoft Foundry agents](https://learn.microsoft.com/azure/ai-foundry/agents/) — LOB, non-LOB y hosted.
- [SharePoint agents](https://learn.microsoft.com/sharepoint/sharepoint-agents) — `.agent files` y librerías documentales.
- [Agent Builder in Microsoft 365 Copilot](https://learn.microsoft.com/microsoft-365-copilot/agent-builder-overview) — agentes declarativos sin pasar por Copilot Studio.
- [Microsoft 365 Agents Toolkit](https://learn.microsoft.com/microsoft-365/agents-toolkit/) — extensión de VS Code para agentes pro-code.

### Work IQ MCP servers

- [Tooling servers overview (Work IQ MCP)](https://learn.microsoft.com/microsoft-agent-365/tooling-servers-overview) — los 8 servidores y qué dan acceso.

### SDKs

- [Microsoft 365 Agents SDK](https://learn.microsoft.com/microsoft-365/agents-sdk/) — transporte conversacional. npm: `@microsoft/agents`.
- [Microsoft Agent 365 SDK](https://learn.microsoft.com/microsoft-agent-365/sdk/) — gobernanza, observabilidad, MCP. npm: `@microsoft/agent365`.
