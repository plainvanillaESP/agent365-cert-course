## Estructura de cada módulo

Cada módulo tiene la siguiente estructura de archivos:

- `README.md` — overview del módulo, duración, objetivos de aprendizaje, validación
- `teoria.md` — contenido teórico con sub-secciones temporizadas
- `laboratorios.md` — laboratorios prácticos con prerrequisitos y troubleshooting
- `evaluacion.md` — preguntas oficiales del banco con justificación y caso de estudio
- `recursos.md` — referencias a documentación oficial Microsoft Learn
- `assets/` — capturas, diagramas SVG, recursos visuales

## Convenciones

- Cero emojis en cualquier archivo (regla absoluta).
- Iconos sólo en el shell React, vía Lucide o Material UI.
- Citas a fuentes oficiales con la sintaxis `[texto](URL)` de Microsoft Learn.
- Badges de estado de capacidades del producto: `[GA]`, `[Preview]`, `[Frontier]`, `[Deprecated]`.

Detalle completo en [`docs/convenciones-redaccion.md`](../docs/convenciones-redaccion.md).
