# Módulo 16. Costes, optimización y mejores prácticas

**Duración estimada:** 45 min lectura + 12 min quiz + 60 min labs = 117 min total.
**Área del examen:** 5 (Monitor, troubleshoot and improve).
**Estado:** Producido (Fase E.10).

Aporta **1 pregunta al banco oficial** del examen. Con M16 cerrado, el banco queda al 100 por ciento (60/60).

## Archivos del módulo

- [`teoria.md`](./teoria.md). Contenido teórico (5 secciones, 45 min lectura).
- [`quiz-practica.md`](./quiz-practica.md). 5 preguntas Q-16-1..Q-16-5 más caso de estudio (Mapfre).
- [`laboratorios.md`](./laboratorios.md). 2 labs prácticos (60 min total).
- [`recursos.md`](./recursos.md). URLs de Microsoft Learn y frameworks (FinOps, TBM, ITIL).
- `module.yaml`. Manifiesto del módulo con 5 OAs, glosario y secciones.

## Objetivos de aprendizaje

1. Construir el modelo TCO con cuatro líneas canónicas (licencias, ingestión, storage, operación).
2. Optimizar licencias Copilot 365 con la regla de cuarentena 30/30.
3. Optimizar el catálogo de agentes con el criterio de retirada por uso trimestral.
4. Aplicar tiering de retención canónico (30 días caliente, hasta 7 años frío, glacial opcional).
5. Operar el ciclo de mejora continua trimestral con cuatro inputs y tres outputs canónicos.

## Prerrequisitos

- M03, M10, M11, M12, M13, M14.
- Familiaridad con licenciamiento Microsoft 365 y reporting financiero básico.
